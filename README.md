# Smart Quiz App

Preprosta osnova za projekt "Smart Quiz App" (React + Vite).

## Zagon

```bash
npm install
npm run dev
```

Lahko uporabis tudi:

```bash
npm start
```

## Demo admin prijava

- Email: `admin@smartquiz.app`
- Geslo: `admin123`

## Vsebina

- registracija in prijava uporabnikov
- zasciten dostop do dashboard/profil strani
- resevanje kvizov
- pregled rezultatov in osnovne statistike
- urejanje profila
- admin CRUD za kvize
- admin dostop samo za admin uporabnika
- odjava in prikaz napak na obrazcih
